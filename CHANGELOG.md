# Changelog

All notable changes to the "terminal-error-sound" extension will be documented in this file.

## [1.0.0] - 2024-01-01
### Added
- Initial release
- Terminal error detection using onDidWriteTerminalData
- Cross-platform sound playback (Windows, Mac, Linux)
- Custom sound path support
- Volume control
- Enable/disable toggle command
- Test sound command
- Configurable error regex patterns
